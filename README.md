# 🌿 Mood Journal

A simple Flask web application that allows users to write and track their daily moods.

---

## 🚀 Live Demo
https://your-app-link.onrender.com

---

## 📌 About the Project
Mood Journal is a beginner-friendly web application built using Flask. It helps users record their daily mood in a simple and clean interface.

This project was created for practice to learn:
- Web development using Flask
- Git & GitHub workflow
- Cloud deployment using Render

---

## 🛠️ Tech Stack
- Python 🐍  
- Flask 🌶️  
- HTML / CSS  
- Gunicorn (production server)  
- Render (hosting platform)

## ⚙️ How to Run Locally

git clone https://github.com/Thrisha83/moodjournal.git  
cd moodjournal  

python -m venv venv  
venv\Scripts\activate   # Windows  

pip install -r requirements.txt  

python app.py
